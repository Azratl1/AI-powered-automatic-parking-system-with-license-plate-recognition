# Разработка системы автоматической парковки с распознаванием номерных знаков на основе технологии ИИ

Интеллектуальная система автоматической парковки, предназначенная для автоматизации контроля доступа транспортных средств на парковочную территорию с использованием методов компьютерного зрения и технологий искусственного интеллекта.

---

## Описание проекта

Проект направлен на разработку системы автоматической парковки с функцией распознавания государственных регистрационных знаков транспортных средств. В основе решения используются методы глубокого обучения и обработки изображений для обнаружения транспортных средств, идентификации номерных знаков и автоматической регистрации событий въезда и выезда.

Разрабатываемая система ориентирована на повышение эффективности управления парковочным пространством, сокращение времени обработки транспортных потоков и снижение влияния человеческого фактора.

---

## Предмет исследования

Процесс автоматизации парковочных операций с использованием методов глубокого обучения для распознавания номерных знаков.

---

## Объект исследования

Технологии разработки системы автоматической парковки на основе модели глубокого обучения с использованием современных средств компьютерного зрения.

---

## Цель проекта

Создание системы автоматической парковки с функцией распознавания номерных знаков для автоматизации контроля доступа и учёта времени пребывания транспортных средств.

---

## Возможности

-автоматическое обнаружение транспортных средств и номерной раки;
-распознавание государственных регистрационных знаков;
-фиксация времени въезда и выезда;
-обработка данных в режиме реального времени;
-поддержка дальнейшего расширения функциональности.
